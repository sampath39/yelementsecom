Frontend:

* Next.js (App Router)
* React
* Tailwind CSS

* Lovable Cloud (for DB, Auth, APIs)

1. FRONTEND UI

* Design similar to Dmart:

  * Green header
  * White background
  * Yellow highlights
* Clean grid layout
* Fully responsive

Pages to implement:

* Home Page (banners, categories, offers)
* Category & Subcategory Pages
* Product Listing Page (filters + search)
* Product Detail Page
* Cart Page
* Checkout Page
* Login Page
* Register Page
* User Dashboard

* Vendor

* Dashboard
* Admin panel
* Vendor panel
